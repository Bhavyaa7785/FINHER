Project Description

FINher is a personal budget tracking and financial management application specifically designed to empower women. This easy-to-use platform allows users to:

Track daily expenses

Monitor investments

Set and achieve financial goals

Additionally, the app generates downloadable PDF reports and offers financial education resources. It is built using Python and Streamlit, providing a lightweight and user-friendly financial solution.
